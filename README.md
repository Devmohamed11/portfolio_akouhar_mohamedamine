# Tableau de Bord Portfolio - Mohamd Amine Akouhar

Un tableau de bord portfolio personnel moderne et interactif construit avec **HTML5, CSS3 et JavaScript vanilla**. Présente une interface élégante en mode sombre avec des effets de glasmorphisme, des animations fluides et un design entièrement réactif.

## 🎨 À Propos du Logo

Le logo **"MA"** affiché dans la barre latérale est une **abréviation élégante et personnalisée extraite du nom complet "Mohamed Amine"**.

### Conception du Logo
- **Initiales**: Les lettres "M" et "A" représentent respectivement "Mohamed" et "Amine"
- **Dégradé Sophistiqué**: Le logo utilise un dégradé bleu-violet (#3B82F6 → #8B5CF6) qui s'harmonise parfaitement avec la palette de couleurs générale
- **Centrage Professionnel**: Le logo est centré dans la barre latérale pour une présentation équilibrée
- **Taille Optimisée**: 60x60px sur les écrans de bureau, 50x50px sur les appareils mobiles
- **Identité Visuelle**: Crée une connexion personnelle et mémorable avec le portfolio

## 🎨 Caractéristiques de Conception

### Conception Visuelle
- **Interface en Mode Sombre**: Fond bleu marine profond (#0F172A)
- **Glasmorphisme**: Effet de verre dépoli avec flou de toile de fond
- **Accents Dégradés**: Dégradés bleu-violet pour la profondeur visuelle
- **Animations Fluides**: Effets de survol, transitions et éléments flottants
- **Mise en Page Réactive**: Approche mobile-first avec points d'arrêt adaptatifs

## 📋 Fonctionnalités

### 1. Section Tableau de Bord
- **Introduction Héroïque**: Accueil audacieux avec emoji d'onde animée
- **Boutons d'Appel à l'Action**: Boutons "Voir les Projets" et "Me Contacter"
- **Logo Centré**: Logo "MA" élégant dans la barre latérale
- **Navigation Fluide**: Transitions douces entre les sections

### 2. Section Projets
- **Grille de Projets**: Mise en page réactive 3 colonnes (bureau), 1 colonne (mobile)
- **Cartes de Projets avec Images**:
  - Images authentiques des projets
  - Titre et description
  - Balises technologiques
  - Bouton "Voir les Détails"
- **Fonctionnalité de Recherche**: Recherche de projets en temps réel
- **Boutons de Filtre**: Filtrer par catégorie (Tous, Web, App)
- **État de Chargement**: Spinner animé

### 3. Modal Détails du Projet
- **Arrière-plan Superposé**: Arrière-plan sombre flou
- **Contenu Modal**:
  - Image du projet
  - Titre du projet
  - Description complète
  - Technologies utilisées
  - Bouton "Visit Website"
- **Animations Fluides**: Entrée glissante vers le haut
- **Support Clavier**: Fermer avec la touche Échap

### 4. Section Contact
- **Formulaire de Contact**:
  - Entrée du nom
  - Entrée d'email
  - Zone de texte du message
  - Bouton d'envoi
- **Message de Succès**: Confirmation après l'envoi
- **Validation du Formulaire**: Validation HTML5

### 5. Navigation
- **Navigation de la Barre Latérale**:
  - Barre latérale fixe à gauche (toujours visible)
  - Indicateur d'état actif
  - Logo "MA" centré et élégant
- **Transitions de Section Fluides**: Animations de fondu entrant
- **Responsive**: S'adapte automatiquement à tous les appareils

## 🛠️ Pile Technologique

### Technologies Frontend
- **HTML5**: Balisage sémantique et structure
- **CSS3**:
  - Mises en page Flexbox et CSS Grid
  - Animations et transitions CSS
  - Filtres de toile de fond pour le glasmorphisme
  - Variables CSS pour le thème
  - Media queries pour la réactivité
- **JavaScript (Vanilla)**:
  - Manipulation du DOM
  - Gestion des événements
  - Rendu de contenu dynamique
  - Gestion modale
  - Gestion des formulaires

### Bibliothèques Externes
- **Font Awesome**: Bibliothèque d'icônes (v6.4.0)
- **Google Fonts**: Polices Poppins et Inter
